# Labs-Prog-Estat

Esse repositório é referente as atividades computacionais da matéria de programação estatística, do curso de estatística da UFSCar, ministrada pelo professor Dr. Thiago Rodrigo Ramos do departamento de estatística, as atividades são referentes a simulação de distribuições de probabilidades e estimação via monte carlo e importance sampling.

Os enunciados das atividades podem ser encontrados no material do professor Dr. Thiago Rodrigo Ramos via link: https://github.com/thiagorr162/metodos_computacionais/tree/main/notebooks/grad
